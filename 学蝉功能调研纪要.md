# 学蝉功能调研纪要

> 调研时间：2026-03-27
> 调研对象：学蝉 SaaS 平台（机构端管理后台）
> 目的：为全民体育文化平台（机构端）提供功能参考

---

## 📋 核心功能模块

### 一、学员管理
| 功能 | 说明 | 优先级 |
|------|------|----------|
| 学员列表 | 搜索、筛选、标签分组 | P0 |
| 学员档案 | 基本信息、体质数据、成长记录 | P0 |
| 报名记录 | 报名课程/班级、缴费状态 | P0 |
| 考勤记录 | 缺勤、请假、迟到统计 | P0 |
| 课时消费 | 课时余额、消费明细 | P0 |
| 成长档案 | 体质变化、技能进步、作品展示 | P1 |

### 二、教练管理
| 功能 | 说明 | 优先级 |
|------|------|----------|
| 教练列表 | 基本信息、资质认证、评分 | P0 |
| 教练排课 | 个人课表、上课时间设置 | P0 |
| 教练评价 | 学员评分、反馈汇总 | P1 |
| 教练数据 | 上课次数、好评率、收入统计 | P1 |

### 三、课程管理
| 功能 | 说明 | 优先级 |
|------|------|----------|
| 课程列表 | 课程信息、价格、上架状态 | P0 |
| 班级管理 | 创建班级、分配教练、设置名额 | P0 |
| 排课管理 | 时间表、教室、上课安排 | P0 |
| 课时设置 | 每课时费、包课价格、优惠策略 | P0 |
| 试听课程 | 免费试听、转化统计 | P1 |

### 四、订单管理
| 功能 | 说明 |  
|------|------|
| 订单列表 | 订单号、学员、课程、金额、状态 | P0 |
| 订单详情 | 支付方式、支付时间、发票信息 | P0 |
| 订单导出 | Excel 导出、财务对账 | P1 |
| 退款管理 | 退款申请、审核、退款处理 | P1 |

### 五、财务管理
| 功能 | 说明 | 优先级 |
|------|------|----------|
| 收支明细 | 收入、支出、时间筛选、分类筛选 | P0 |
| 收款统计 | 日/周/月/年汇总、趋势图 | P0 |
| 合同管理 | 合同模板、签署记录、电子合同 | P1 |
| 审批管理 | 退款/调课/退课审批流 | P1 |
| 课消金额 | 课时自动扣减、对账 | P0 |
| 工资管理 | 教练课时费、提成计算、发放记录 | P1 |

### 六、家校互动
| 功能 | 说明 | 优先级 |
|------|------|----------|
| 课后点评 | 教练给学员写评价、评分 | P0 |
| 课程通知 | 上课提醒、调课通知、活动通知 | P0 |
| 消息推送 | 系统消息、课程变更、缴费提醒 | P0 |
| 反馈收集 | 学员/家长反馈处理 | P1 |

### 七、招生管理
| 功能 | 说明 | 优先级 |
|------|------|----------|
| 意向学员 | 潜在客户线索、来源追踪 | P1 |
| 跟进记录 | 电话/微信跟进、跟进阶段 | P1 |
| 试听记录 | 试听反馈、转化结果 | P1 |
| 转化漏斗 | 从咨询到签约的转化率 | P2 |

### 八、系统设置
| 功能 | 说明 | 优先级 |
|------|------|----------|
| 机构信息 | 名称、Logo、地址、联系方式 | P0 |
| 账号管理 | 管理员角色、权限分配 | P0 |
| 标签体系 | 学员标签、课程标签自动归类 | P1 |
| 数据备份 | 手动备份、自动备份、恢复 | P1 |

## 🎯 MVP 优先级划分

### 第一期（必须）
- 学员列表 + 档案
- 教练列表
- 课程列表 + 班级管理
- 排课管理
- 订单列表
- 收支明细
- 课后点评

### 第二期（重要）
- 课时消费
- 成长档案
- 教练排课 + 数据统计
- 订单导出
- 合同管理 + 审批管理
- 课消金额
- 工资管理
- 家校互动（通知/推送）

### 第三期（增强）
- 意向学员 + 跟进 + 试听
- 退款管理
- 转化漏斗
- 标签体系
- 数据备份

---

## 💡 核心差异点

| 学蝉 | 全民体育文化平台 |
|------|----------------|
| 通用培训行业 | 聚焦体育培训 |
| B端机构管理 | B端 + C端 + 赛事服务 |
| 无赛事功能 | 赛事IP、体测处方、证书颁发 |
| 纯SaaS | 轻资产运营（场地/教练不自有） |
